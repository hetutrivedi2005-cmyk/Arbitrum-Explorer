How Proof of Work Works

A simple guide explaining the cryptographic components of the mining simulation sandbox above.

What is a Hash?

A cryptographic mathematical algorithm (like SHA-256) that maps any input data into a fixed-length string of 64 hexadecimal characters. It is a one-way function—impossible to reverse.

What is a Nonce?

Short for "Number used once". Since changing the block data itself is not allowed, miners change this arbitrary number field to yield completely different cryptographic hash outputs.

Proof of Work

The process of searching for a nonce value that yields a hash beginning with a specific number of zeros. Finding a valid hash is difficult, but verifying it is instantaneous.

Immutability

Because each block references the previous block's hash, modifying data in any single block breaks all subsequent link connections in the chain, preventing malicious history changes.
